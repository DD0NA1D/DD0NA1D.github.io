甘加草原之旅

摘要

2021年7月31日至8月1日两天。与父亲、父亲的朋友、与我的朋友一起从兰州前往甘南的甘加草原。

第三天，在父亲的建议下。我与朋友去玩了两个多小时桌游，晚上和他们与他们的家长一起吃饭。在这里一起记录。

Day 1：7月31日

草原的景色

我们调侃说草原美的像是Windows XP的桌面。

菠菜 ——带来惊喜的金毛

父亲的同学温x叔叔带来了一条温顺的金毛，叫菠菜。她不反对大家摸她的头，但似乎也不期待。但有时她很享受，我手累把手架在扶手上，她会站起来方便我摸


菠菜让我一点都不害怕狗了（至少是金毛），或许养条狗也不错，等我爸妈退休后养条金毛或者边牧也许也挺好。

吃饭的时候她会自己钻到桌子地下趴着，好可爱

农家乐的游戏闲谈时间

我和父亲选了最大的房间，大家都来我们的房间玩。

大家是分两批来的，我所在的第一批是下午到的。

下图是大家在边玩边等第二批人，有些人在睡觉没来。（我的脚边是菠菜）大家玩数七数青蛙喝酒吃菜。有意思的消遣。

温x叔叔在某大学教书，说教的是“未来科学”。他是个善于表达、喜欢表现自我的人。他推测未来游戏的趋势是VR×AR之类的。想起父亲当年挖比特币也是和他一起，这是有趣的预测。以后进游戏行业也可以看看这个方向。

练车

农家乐的游戏玩腻了，大家提出开车去白石崖溶洞看看。

我7月12号才拿了驾照，正处在练车的狂热期。父亲喝了点酒，我建议我来开，大家同意了。

大家的包容令我很感动，因为带上一个新手会大幅度拖慢整个车队的速度——从100码上下拖到50码左右。吕h阿姨头车，一直很照顾我。

我们慢慢开着，也欣赏着沿途的景色。甘加草原的一大特色是：路边有很多可爱的土拨鼠

拍星星比赛与震撼的星空

凌晨，大家一起吃完晚饭，穿上厚衣服，到农家乐外面看星星。看星星最好是在伸手不见五指的黑暗中看，这样才能看清楚。

我这辈子都没见过那么多星星。看着那漫天的星星，对伟大自然的敬畏、对神秘星空的向往一时涌上。那时，我想，人生是为了一些超越平凡琐碎的更美丽的东西，也有些风景只能体验而不能保留。

王x叔叔带了天文望远镜，可惜他不会调。能看上一眼天文望远镜应该会很不错。

父亲提议大家来个拍星星比赛。

做过功课的就是不一样，拍星星时，如果用默认相机根本拍不清楚。父亲之前看了教程，就教我们用专业模式拍，还教了我们参数设置。要调白平衡、ISO、快门速度。温x叔叔显然之前也有所研究：他教我们打开语音拍照——这样就可以把手机放在草地上，人离开后喊一声拍照。

大家的设备、参数、和后期都有区别，导致同一片星空拍出来也不同。

Day 2：8月1日

日出与清晨的驾车探索

第二天五点半，大家开始起床看日出。我起晚了，起来的时候去观景台的人已经走了，我们就在农家乐门口看。虽然太阳还没翻上山，但天已经大亮了。所以我认为草原的日出并没有什么特别的。

早餐8:30才能吃，看完日出还有很多时间。我爸决定带我去练练车，就往白石崖溶洞开。开了很长的上山路，和我爸闲聊一路。最后停在一个山上的观星营地，从山上拍了一个远处的城镇。安静的山上、初升的太阳、早晨的雾、村里的炊烟。由于我开的慢，我爸为了准时赶上早饭，拍张照就拉着我走了。我只匆匆拍了张照。

白石崖溶洞的「奇妙冒险」

进溶洞之前，我以为是以观景为主。进去以后才发现，攀岩才是最大的乐趣。我们总结：进去前还怕弄脏手，出来以后但凡能用上手的地方绝不只用脚。

溶洞的最深处只是一个绿色的很小的湖，被当地藏族人奉为圣水，不能洗手，也不好看。从洞里出来后，我们整个手和衣服裤子上都是泥巴。

在河边野餐

小插曲：孙队的无人机没电坠机了

孙队长的无人机坠机了，大家上山去找。山很陡。寻找的过程又是四肢并用、很刺激。我才发现健身的重要性、已经我可能有些恐高。最后在计算费用的时候，大家很贴心地免去了孙队的费用，约五百元左右。我感觉很暖心，希望我以后也能有这样的朋友。

Day 3: 与朋友的桌游

8月1日，在父亲的建议下，我在A1高闪来一个（之前几次活动一直扩大的微信群，由父亲同学们的孩子们组成）里组了一场桌游。（这个暑假跟成都的同学桌游玩的比较多）

大家的反应还不错，算是比较积极。除了胡yy与王ry，大家都参与了讨论，有三个或者四个不在兰州的，一个要上课的。

最后刘jy、牛zy、汪zx与我一起在凤鑫大厦上的大侦探玩了桌游。我很享受游戏的过程，场面没有我想象的尴尬，我认为这是一个良好的开始，以后回兰州可以喊这些朋友们一起玩了——希望这个群也能活跃起来，像他们的父辈一样：但这并不容易，大家不是同学，一年基本上最多见两次。一年一两聚应该也是很好。

我们在玩达芬奇密码，左下角涂掉的是工作人员

我们在玩宝石商人，挺有趣的游戏，最后刘jy领先我一分赢了。（牛zy说她更喜欢玩密室，看来初中生和我们的爱好不太一样）

写在最后

这篇日志是我建站以来的第一篇生活日志，这具有非凡的意义。

生活中的那些瞬间，当时的感受无法复刻，如果用我不熟悉的英文记述肯定会有第二次损失 ——所以更希望用中文写所有的生活类博客（这也是为什么分类标签是拼音“生活”）。此外，网络表情也与网络用语也是我所希望避免使用的。只希望在某天重新打开日志的时候，至少能拾得一点两点当时的感受。此外，我也想在生活类博客中尽可能避免任何与技术相关的东西。人活着是为了生活，技术只是帮我们活着。

写这篇日志的时候，一直在纠结一个问题：我的网站要不要公开？如果公开，我将有很多不能写的东西：比如朋友的全名、职业、住所，不便于写的事。我也将有很多不能上传的图片与视频：任何可能泄露朋友信息的内容。

说到底，博客与日记的区别在哪里？我所写的博客，是给谁看？我自己？还是所有人？

传统的博客，是给所有人看的。而我想写的东西，至少生活博客，是给自己看的——也许在某一天，我也会想将一部分内容分享给其他人。从互联网上消失也很简单——取消在github.io上的部署。但万一呢？万一哪天我存在本地的日记被人拷走，或者被什么黑客或者病毒偷走了呢？“正经人谁写日记？”毫不保留地将一切写在纸上，就算放在世界上最安全的地方，也会隐隐不安。不加修饰的内容，一经泄露，也会伤害很多人吧 (所以我们有时需要善意的谎言)。所以，写生活类博客，还是遵循“在保证安全的前提下，最大限度地坦诚”。

How to Add Live2d model to Hexo?

Basic Version: Model and Sound only (default).

  1. Install hexo-help-live2d

  1. Select and download a well-prepared model

    Preview
    Prepared packages

  2. Set Model Asset Directory

    1. create a folder under root path of blog: ./live2d_models
    2. create a folder: ./live2d_models/{model_name}
    3. copy files from ./node_modules/live2d-widget-model-{model_name}
      • /exp
      • /moc
      • /mtn
      • /snd
      • *.model.json
      • *.physics.json
      • *.pose.json
      • [P.S.] Some DIRs/Files are optional
    4. paste them to ./live2d_models/{model_name}

  3. IMPORTANT Solution to EISDIR: Fix hexo-help-live2d

    For unknown reasons, downloading live2d packages might alter some configurations or settings of hexo-help-live2d. Error EISDIR(short for Error is Dir) would launch. Plugin hexo-help-live2d load failed.

    One solution( which works well for me) is to reinstall hexo-helper-live2d. Don’t worry, reinstallation takes less time.
