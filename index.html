How to Add Live2d Kanban girl model to Hexo?

Basic Version: Model only.

  1. Install hexo-help-live2d

  1. Select and download a well-prepared model

    Preview
    Prepared packages

  2. Set Model Asset Directory

    1. create a folder under root path of blog: ./live2d_models
    2. create a folder: ./live2d_models/{model_name}
    3. copy files from ./node_modules/live2d-widget-model-{model_name}
      • /exp
      • /moc
      • /mtn
      • /snd
      • *.model.json
      • *.physics.json
      • *.pose.json
      • [P.S.] Some DIRs/Files are optional
    4. paste them to ./live2d_models/{model_name}

  3. IMPORTANT Solution to EISDIR: Fix hexo-help-live2d

    For unknown reasons, downloading live2d packages might alter some configurations or settings of hexo-help-live2d. Error EISDIR(short for Error is Dir) would launch. Plugin hexo-help-live2d load failed.

    One solution( which works well for me) is to reinstall hexo-helper-live2d. Don’t worry, reinstallation takes less time.
